üßÆ Finite Difference Methods in Heat Conduction

üìò Introduction

Finite Difference Method (FDM) is a powerful numerical technique used to approximate solutions to differential equations, especially useful for analyzing heat conduction problems where analytical solutions are difficult or impossible.

The heat conduction equation is replaced by discrete algebraic expressions over a grid of points, enabling computational solutions through iterative or direct methods.

üßæ Governing Equation

The 1D steady-state conduction without heat generation is given by:

\[ \frac{d^2T}{dx^2} = 0 \]

Using the central difference approximation, this becomes:

\[ \frac{T_{i+1} - 2T_i + T_{i-1}}{\Delta x^2} = 0 \]

This equation forms the basis for calculating temperatures at interior nodes of a solid using a grid-based approach.

üßÆ Discretization and Node System

Consider a rod of length \( L \) divided into \( N \) nodes. Each node \( i \) represents a spatial location where temperature \( T_i \) is calculated. With boundary temperatures known at the ends, the system of equations can be solved to find intermediate values.

For example, if the boundary conditions are \( T_0 = 100^\circ C \) and \( T_4 = 50^\circ C \), then the temperatures at internal nodes \( T_1, T_2, T_3 \) can be computed by solving a tridiagonal system.

üí° Extension to Unsteady State

For transient problems, the governing equation is:

\[ \frac{\partial T}{\partial t} = \alpha \frac{\partial^2 T}{\partial x^2} \]

Using finite difference in time and space:

\[ \frac{T_i^{n+1} - T_i^n}{\Delta t} = \alpha \frac{T_{i+1}^n - 2T_i^n + T_{i-1}^n}{\Delta x^2} \]

This explicit method updates temperature at each time step using previous values. Stability depends on the Fourier number \( Fo = \frac{\alpha \Delta t}{\Delta x^2} \).

‚öôÔ∏è Key Concepts

  • Grid spacing (\( \Delta x \)) and time step (\( \Delta t \)) critically affect accuracy and stability
  • Explicit methods are easier to implement but conditionally stable
  • Implicit methods (like Crank-Nicolson) are unconditionally stable but require solving equations at each time step
  • Used extensively in simulations involving complex geometries and materials
